# Logs Directory

This folder stores runtime logs generated while running the application locally.

## What goes here

- Application logs configured through Spring Boot (for example, `pokedex.log`)
- Console redirection logs created during manual runs (for example, `app-startup.out.log`)

## Notes

- Log files are ignored by Git through `.gitignore`.
- Keep this file tracked to document the folder purpose.
- You can safely delete log files before sharing the project.
